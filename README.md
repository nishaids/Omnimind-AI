## 🚀 What is OMNIMIND AI?
Scan any company and instantly get AI-powered:
- 🔬 Company Research & Intelligence
- 📈 Live Stock Analysis
- 📰 News Intelligence & Sentiment
- 📊 Executive Report
- 📄 PDF Export & Email Delivery

## 🛠️ Tech Stack
- Python, Streamlit, Groq AI
- yfinance, NewsAPI, ReportLab
- 4 Autonomous AI Agents

## ⚙️ How to Run Locally
```
pip install -r requirements.txt
python main.py
```

```

```
